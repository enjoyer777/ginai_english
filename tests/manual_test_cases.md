# Ручные тест-кейсы — приёмка по разделу 6 BR

Deliverable #8. Заполняется заказчиком/исполнителем при приёмке.

**Окружение:** `docker compose up -d --build` на VPS, бот доступен по @username, Я.Таблица загружена, Bitrix24 webhook настроен, чат менеджеров создан.

## Что сделано до начала ручных тестов (юнит-тесты)

```bash
pytest tests/unit -q
```

30 юнит-тестов закрывают: рабочие часы (граница 19:00 эксклюзивна, выходные), guardrails (jailbreak/мусор), сборку payload Bitrix24, парсинг xlsx (валидация направлений/уровней, корректная типизация installment, наличие настроек).

Юнит-тесты — про корректность кода. Этот документ — про корректность поведения с точки зрения BR.
